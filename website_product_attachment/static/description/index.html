Website Product Attachments

Website Product Attachments module allows admin uploading files and attachments and allow to download them to relevant product.


Website Product Attachments app provides functionality for file downloads available within product pages from admin.Admin can upload product manuals, guides, specification document , etc. Attachment can be associated with individual products.



All attached attachment display in website product page.



Admin can enable / disable using customize option.
