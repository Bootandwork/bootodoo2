Web Push Notifications

OneSignal

Create OneSignal Account

This module allows to sends personalized push notifications to subscribed users from websites via OneSignal account app of website.

Configuration in Odoo

Website > Configuration > Setting > OneSignal Web Push Notifications

Successful Configuration

You can see popup in your website that ask users for enable push notifications.
