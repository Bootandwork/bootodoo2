Detailed Features List

Below is the list of features of Odoo Power BI Connector.

Create or Export Datasets

  • You can create datasets with one or more tables.
  • Moreover, you can also export datasets to powerbi.

Multi Workspace Support

  • odoo Power BI Connector supports many workspaces.
  • You can also create workspaces in Odoo and export them.

Filters for the Ease to Search Data

  • You can use filters to distinguish data from tables.
  • For example, you can filter products based on the selected model fields.

Ease to export data from Odoo.

  • You can export data from any Odoo model by creating tables.
  • You can configure cron to schedule data export for any model/table.

Managing Dynamic Server Actions

  • You can generate dynamic server actions for any model using datasets.
  • It helps you to export the data from the report of that model to the Power BI tool.

Track and Analyze data from the dashboard

  • Import reports and dashboards from powerbi.
  • You can also visualize these reports and dashboards from Odoo.

Provide Table Relationship

  • New You can use Table Relationships to relate the different tables.
  • For example, Create a table for sale. order model with partner ID res. partner in one column, validate it and add it to the dataset for data export to powerBi.
